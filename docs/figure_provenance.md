# Figure provenance

- Main and supplementary PNG charts are generated by
  `src/pm25forecast/viz.py`.
- Source data are the processed and result CSV files.
- PNG files must not be manually edited.
- The conceptual architecture is maintained in the editable source
  `paper/figures/00_framework.drawio`.
- `scripts/export_framework_pdf.py` creates the matching
  `paper/figures/00_framework.pdf` used by the manuscript. The source remains
  editable in Draw.io and every arrow must agree with the implemented code.
