# PM2.5 Research Dashboard

This app follows the organization style of the provided DAP391m sample
dashboard, but it reads this repository's UK-Air PM2.5 data and generated
research artifacts.

## Structure

```text
app/
├── streamlit_app.py
├── components/
│   ├── overview_tab.py
│   ├── performance_tab.py
│   ├── figures_tab.py
│   ├── chatbot_tab.py
│   └── about_tab.py
├── pipeline/
│   └── data_pipeline.py
└── visuals/
    └── charts.py
```

## Tabs

| Tab | Purpose |
|---|---|
| Dataset Analytics | Shows station manifest, raw station preview, PM2.5 coverage, and daily pattern |
| Performance Dashboard | Shows five-station/five-seed results, MAE curves, gate behavior, and matched effectiveness |
| Generated Figures | Displays figures generated by `scripts/make_figures.py` and paper figures |
| Research Q&A | Static explanation assistant for presentation/demo |
| About | Explains the research question, method, scope, and interpretation |

## Run

From the repository root:

```powershell
streamlit run app/streamlit_app.py
```

The app does not retrain models and does not invent demo metrics. Missing
panels mean the corresponding result file has not been generated yet.
