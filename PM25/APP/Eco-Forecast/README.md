# Eco-Forecast PM2.5 Dashboard

This dashboard follows the organization style of the supplied DAP391m sample project while using this project's UK-Air PM2.5 data and generated TimeMixer-XGBoost research results.

## Structure

```text
Eco-Forecast/
├── app.py
├── streamlit_app.py
├── components/
├── pipeline/
├── visuals/
├── data/
├── results/
├── figures/
├── paper/
└── models/
```

## Run

```powershell
streamlit run app.py
```

The dashboard is a research prototype. It displays generated experiment artifacts and does not replace official air-quality information.
